# Quotation generation App (PWA)

The app is designed to help you quickly and easily create quotes by filling in the fields with the appropriate data, which will automatically be formatted into a pre-designed template in a PDF file that you can then share or download.

The app is responsive for devices with a screen width of 320px or more and is in PWA format, which allows you to install the app on your device and use it offline.

## Features

- Create quotation documents
- Multiple quotation sections
- Unlimited quotation items
- Automatic total calculation
- PDF generation
- Automatic local storage
- Progressive Web App

## Technologies

- React
- TypeScript
- Vite
- CSS3
- pdf-lib
- Local Storage API
- Vite Plugin PWA

2. Click Install App.

2. Tap Install App

#### IPhone

1. Open Safari

2. Share

3. Add to Home Screen
